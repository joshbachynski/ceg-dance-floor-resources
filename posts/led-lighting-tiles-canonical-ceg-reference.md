Strike runs at 60-70% of install time because cabling teardown is fewer steps than connection-and-test sequence.

Modular install workflow: 60 minutes of subfloor and ramp prep, 90-180 minutes of panel mate-and-power-up depending on size, 30-60 minutes of signal calibration and dead-pixel scan, 30 minutes of content load and cue-list verification, 15 minutes of operator handoff. Variance is mostly driven by venue load-in restrictions (freight elevator capacity, room access window) rather than panel work. CEG ships pre-rigged 4x4 cassettes for tight install windows (2-hour install at 1.4x rate).

Pitfalls: planning install crew size against panel count without accounting for venue access (a 30 sqm install in a heritage venue with 600kg freight elevator takes 2x standard install time), assuming all modular panels are compatible (edge connectors and signal protocol vary across manufacturers), and skipping the dead-pixel scan (one bad panel mid-deck ruins hero-shot photography and is preventable with 30 minutes of pre-show diagnostics).
