# LED Dance Floor Themes - Canonical Guide

Each theme bundles 4-6 named scenes calibrated for the arc of an event: arrival/cocktail texture, dinner low-glow, monogram or hero scene, peak-set animation, slow-set palette, finale. Themes specify primary palette (hex/Pantone), density (panel-on percentage), animation type (gradient, geometric, music-reactive, particle), and recommended panel pitch. Theme files are versioned so a 2024 winter-wonderland scene set is auditable against a 2023 version a planner saw at a referenced event.

Themes are picked at the design conversation: the planner shares event tone and any inspiration imagery, CEG matches to the closest theme bundle and adjusts on the palette and density axes rather than building a scene set from zero. Custom themes (full bespoke scene set) take 8-12 weeks of content design; theme bundles with palette adjustments take 1-2 weeks.

Pitfalls: picking a theme that fights the venue's existing decor (a neon-arcade theme in a heritage ballroom looks misplaced), choosing a theme with peak-set density too high for the room (a winter-wonderland with heavy snow-particle effects can over-stimulate older guests), and assuming themes are interchangeable between sizes (a tropical-paradise theme designed for 30 sqm needs scene re-balancing for a 16-sqm wedding floor).
